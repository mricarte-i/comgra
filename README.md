# COMGRA - 2025

Hecho con Vite + Three.js (y TypeScript)

## Como correr el proyecto:

1. Install pnpm:
   ```bash
   npm install -g pnpm
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173` to see the application running.
